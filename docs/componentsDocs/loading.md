# 加载组件

### 示例

<m-loading :isShow="true" text="加载中..." />

### 代码

```html
<m-loading :isShow="true" text="加载中..." />
```

### Attributes

| 参数   | 说明         | 类型    | 是否必要 | 默认值 |
| ------ | ------------ | ------- | -------- | ------ |
| isShow | 显示隐藏组件 | Boolean | false    | -      |
| text   | 文本说明     | String  | ''       | -      |
